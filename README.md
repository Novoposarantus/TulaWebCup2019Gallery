# TulaWebCup2019Gallery
gitHub: https://github.com/Novoposarantus/TulaWebCup2019Gallery
Разработчик: Рыжаков Сергей Александрович студент TSU.

Приложение галереи позволяет просматривать изображения, загруженные пользователями. Выставлять им оценки. 
Фильтровать их по именам, даттам и рейтингам. 
Регистрироваться и авторизироваться на сайте для загрузки изображений и выставления оценок.
При первом входе вы можете посмотреть уже загруженные изображения. Так же вы можете смотреть их в режиме слайд шоу, нажав на них.
Чтобы зарегистрироваться нажмите "Зарегистрироваться" и пройдите простую процедуру регистрации. После чего авторизируйтесь на сайте.
Теперь у вас появилась кнопка загрузить изображения. Если нажать на нее можно перейти на страницу загрузки, 
где можно выбрать картинки с устройства, добавить им имена и загрузить на сайт.
